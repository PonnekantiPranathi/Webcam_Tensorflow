Smart Webcam using TensorFlow.js

This is a useful tool which detects objects of any kind, including regular objects like cats, dogs, and people, and draws a box around the detected object using the webcam.

Wait for the model to load before clicking the button.

Activate your webcam and grant access to it when prompted by the browser in order to perform real-time classification by holding objects up close to the camera. Ensure to click "enable webcam" below before proceeding.
